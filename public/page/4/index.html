Cherry-Picking git cherry-pick [--edit] [-n] [-m parent-number] [-s] [-x] <commit> Selectively merge a single commit from another local branch Example: git cherry-pick 7300a6130d9447e18a931e898b64eefedea19544 Squashing WARNING: “git rebase” changes history. Be careful. Google it. git rebase --interactive HEAD~10 (then change all but the first “pick” to “squash”) squash the last 10 commits into one big commit Conflicts git mergetool Work through conflicted files by opening them in your mergetool (opendiff, kdiff3, etc.
